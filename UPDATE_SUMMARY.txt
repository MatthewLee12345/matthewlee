-----------------------------------------------------------------
CUSTOMIZATION TIPS
-----------------------------------------------------------------

To add more film scores:
- Copy one of the <article class="project-card"> blocks
- Change the YouTube embed URL
- Update the title and description
- Add or remove the award banner as needed

To add more research projects:
- Copy one of the research project cards
- Change the gradient class: research-gradient or research-gradient-alt
- Update title, abstract, and tags
- Add publication link

To change colors:
- Edit CSS variables in styles.css (lines 11-19)
- Change --primary-color for main accent color
